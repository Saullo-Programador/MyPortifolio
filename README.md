# 🚀 Saullo Portfolio

Portfólio profissional desenvolvido em Flutter Web/Desktop/Mobile com foco em desenvolvimento Android, Flutter, Backend e Sistemas Modernos.

## ✨ Features

* UI moderna estilo startup
* Responsividade completa
* Dark Theme
* Glassmorphism
* Hover Animations
* Scroll Animations
* Arquitetura escalável
* Código limpo
* Componentização
* Alta performance

## 🛠 Tecnologias

* Flutter
* Dart
* Kotlin
* Java
* React
* TypeScript
* Firebase
* Spring Boot
* MySQL
* Node.js

## 📱 Plataformas

* Web
* Android
* iOS
* Desktop

## ▶ Executar Projeto

```bash
flutter pub get
flutter run -d chrome
```
